# Dataset - ورزشکاران تیراندازی ایران

این دیتاست شامل اطلاعات ورزشکاران تیراندازی ایرانی است و برای پروژه پایانی **استعدادیابی ورزشی** استفاده می‌شود.

## توضیحات

- تعداد رکوردها: ۲۰ ورزشکار  
- رشته: تیراندازی (تفنگ، تپانچه، تراپ)  
- منابع: ویکی‌پدیا فارسی (لینک‌ها در ستون منابع)

## ستون‌ها

1. **نام کامل**: نام ورزشکار  
2. **جنسیت**: مرد یا زن  
3. **تاریخ تولد**: به صورت شمسی  
4. **سن**: به سال  
5. **محل تولد**  
6. **ورزش**: رشته اصلی  
7. **رویداد/شاخه**: رویداد تخصصی  
8. **قد (متر)**  
9. **وزن (کیلوگرم)**  
10. **مربیان**  
11. **افتخارات/مدال‌ها**  
12. **رکوردها**  
13. **حضور در المپیک/بازی‌های آسیایی**  
14. **منابع/لینک ویکی‌پدیا**

## نکات

- ستون‌هایی که اطلاعات ندارند، خالی گذاشته شده‌اند.  
- لینک منابع مستقیم به صفحات ویکی‌پدیا هر ورزشکار است.  
- فرمت فایل: CSV
