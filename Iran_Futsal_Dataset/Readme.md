# دیتاست بازیکنان فوتسال ایران

این مخزن شامل دیتاستی از بازیکنان فوتسال ایران است که برای تحلیل داده و پروژه‌های یادگیری ماشین جمع‌آوری شده است.

## دیتاست

- **فایل:** `Iran_Futsal_Players.csv`
- **تعداد رکوردها:** ۲۰
- **ستون‌ها:**
  - `نام` : نام کامل بازیکن
  - `سال تولد` : سال تولد بازیکن
  - `سن تقریبی` : سن تقریبی بازیکن
  - `قد (متر)` : قد بازیکن به متر
  - `وزن تقریبی (کیلو)` : وزن بازیکن به کیلوگرم
  - `پست` : پست بازیکن (دروازه‌بان، مدافع، مهاجم، وینگر و غیره)
  - `سن شروع حرفه‌ای` : سن تقریبی شروع حرفه‌ای بازیکن
  - `باشگاه‌ها` : باشگاه‌هایی که بازیکن در آن‌ها بازی کرده
  - `تیم ملی` : اطلاعات حضور در تیم ملی
  - `افتخارات اصلی` : مهم‌ترین افتخارات بازیکن
